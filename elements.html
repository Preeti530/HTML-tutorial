HTML Elements and Attributes

What is HTML tag?

  • A tag is the keyword used inside angle brackets("< >") in HTML.

What is an HTML Element?

  • An element is everything from the start tag to the end tag,including the content inside.

Tags and Element:

  1. The Heading:

    • Headings are tags used to define titles and subtitles on your page.
    • Main Heading : <h1gt;,>, <h2>, <h3& <h4>, <h5>, <h6>.
    • They are crucial for creating a logical hierarchy and outline for your content.
      • <h1>Headind1</h1>
      • <h2>Heading2</h2>
      • <h3>Heading3</h3>
      • <h4>Heading4</h4>
      • <h5>Heading5</h5>
      • <h6>Heading6</h6>


  2. HTML Horizontal Rules :

    • the <hr> tag defines a thematic break in an HTML page and is most often display as a horizontal rule.
    • Basically it is used to separate content in an HTML page.

  3. The Line Break Tag :

    • Line Break tag ( <br> )tells the browser,"stop writing on this line and immediately start on the next one."
    • Purpose of line break tag is "creates a single line break.


  4. List Element :

    • List elements ( <ul>, <ol>, <li>): These tags are used to create lists.
      1. <ul>: An unordered list.It creates a bulleted list.
      2. <ol>:An ordered list.It creates a numbered list.
      3. <li>:A list item. Each item in either type of list must be wrapped in an <li> tag.

  5. The Anchor Tag :

    • The anchor tag <a> is what makes the web " hypertext".
    • IT is used to ctreate a hyperlink to another webpage,a file or a location within the same page.
    • Example:<a href="used link which you want">

  6. The Image Tag :

    • The image tag <img> is a self-closing tag used to embed an image onto your page.
    • The image tag is used to display a visual image.
    • It require to main attributes :
      1. src(source):The path or URL to the image file ,this is mandatory.
      2. alt(alternative text):A description of the image,this is vital for accessibility.
      3. Example:<img src="link of any image" height="fix the height of the image as per required" width=" set the width of the image as per required" alt="name the photo as you want">
