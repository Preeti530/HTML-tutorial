HTML lists and Tables

  1. HTML Lists:

    Our day to day lives often involve the use of lists. For example- when we go shopping ,the bill we receive includes a list of all the items we've purchased.

    Types of Lists-
    1. Unordered list
    2. Ordered list

    Undereded list- Display items using bullets.

    Ordered list-Display items in a numerical sequence and supports various numbering styles.

  2. HTMLTables:

    3. HTML tables allow you to arrange data like text,images and links in rows and columns.You use the <table> tag to start and end a table .

    Syntax of HTML table
    <table> //table content </table>

    Key element of HTML table-

    • <table> : Defines the table itself.
    • <tr> : Used for table rows.
    • <th> : Used for table headings.
    • <td> : Used for table data or cells.

  3. Rowspan and Colspan Attributes

    1. Rowspan :
      If you want a table cell to span multiple rows,you can use the rowspan attribute.

    2. Colspan:
      If you want a table cell to span multiple columns ,you can use the colspan sttribute.
