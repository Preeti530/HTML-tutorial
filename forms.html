Forms and Input

  1. Introduction to HTML form:

    HTML form are essential for collecting user input on web pages. Whether it's a search bar ,a login screen or a multi-field registration form,HTML FORMS paly a key role in web interactions.They enable users to submit data ,which can be processed ,stored or returned by a server.

  2. Why do we use forms:

    Forms serve as the gateway between the user and the server,allowing for dynamic , interactive web experiences.

    They are crucial for tasks such as user authentication,data submission,feedback collection and more .

  4. How to use form controls?

    The <input> tag is commonly used to create form controls.The attributes of this tag define the controls behavior.

    • <input type=" " name=" " value=" ">
      1. The "type" attribute specifies the type of input control(e.g.,text,password,checkbox).
      2. The "name" attribute is used for identifying the controls,especially when the data is sent to the server.
      3. the "value" attribute sets a default value for the controls,which the user can overwrite.


  5. HTML Input Types:

    Input type in HTML forms are the backbone of interactive web applications.They allow users to send information to web servers for various purposes like searching,logging in etc.

    1. Text Input:

      The text input type is the most basics form of input and is widely used for collecting simple text data.

      Example: <input type ="text" name="username" placeholder="enter your username">

    2. Password Input:

      The password input type is similar to the text type but hides the characters entered by the user for security reasons.

      Example: <input type ="password" name="password" placeholder="enter your password">

    3. Radio Buttons:

      Radio buttons are used when you want the user to select only one option from a set of choices.

      Example: <input type ="radio" id="male" name="gender" value=" male ">

    4. Checkbox:

      Checkboxes allow the user to select multiple options from a set.

      Example: <input type ="checkbox" id="subscribe" value="yes">
