Basics of HTML

What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard language used to create and structure the content of a webpage.

  • Hyper Text:This refer to the ability to create links that connect web pages to one another.
  • Markup Language:This means you use "tags" to surround your content ,giving the content meaning and structure.

Features of HTML

  • It is platform-independent.
  • Images,Videos,and Audio can be added to a web page.
  • It is a markup language and not a programming language.

HTML Page Structure

An HTML document is structured using a set of nested tags.Each tag is enclosed within angle brackets(<....>) and act as a container for content or other HTML tags.

HTML Boilerplate


<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>

Description of HTML Boilerplate

  1. DOCTYPE Declaration-

    The <!DOCTYPE html> declaration informs the web browser about the HTML version being used. The latest version is HTML 5.


  2. HTML Root Element-

    The <html> tag is the root element that encapsulates all the content on the page.

    • The </html> tag marks the end of the <html> section.


  3. Head Section-

    The <head> tag contains metadata and links to external resources like CSS and Java Script.

    • The </head> tag marks the end of the <head> section.


  4. Title Tag-

    The <title> tag sets the title of the web page,which is display in the browser's title or tab.


  5. Body Tag-

    The <body> tag contains the visible content of the web page .This is where text,image and other elements go.

    • The </body> tag marks the end of the visible content of the web page.

HTML Id and Classes

  1. The Div (The Generic Box) :

    The div tag is a blocklevel container element , it is used to group other HTML elements together so you can style them with CSS and control them with JavaScript.


  2. What is an Id:

    An ID is an attribute ,a unique identifier assigned to only one HTML element within a page.

    It is often used for unique styling and JavaScript manipulations.


  3. What are Classes:

    The class attribute lets you give the same name to multiple HTML elements. That way you can easily change their look or behavior all at once.
